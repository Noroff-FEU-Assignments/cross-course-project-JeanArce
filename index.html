To secure websites from web security attacks. A secure URL should begin with ”HTTPS” rather than "HTTP". The "s" in "HTTPS" stands for secure, which indicates that the site is using a Secure Sockets Layer(SSL) certificate. This lets you know that all your communication and data is encrypted as it passes from your browser to the website's server. Installation of SSL and security plugins. The information sent from your websites will be encrypted and only the targeted recipients will get it. Always check the website whether the latest security software is updated. Updates contains special security patches and features designed to address new threats and, thus, secure websites from hackers. Use strong passwords.

A strong password not be traceable and has to contain special characters, figures, and letters. You should not use common combinations or words or numbers that may be related to your biography. Use anti malware a type of software program to protect information technology systems and individual computers from malicious software. It scans a computer to prevent, detect, and remove malware. Never click embedded links. A link that uses anchor text to create a link to another web page using the HTTP protocol.
